# MakeUp - Ultra Fast
Ultra fast Minecraft shader (Java). For a small graphic improvement. Intended for very low-spec computers.

**Version: 1.5**

## Main features
* Antialiasing (FXAA)
* Depth of field
* Enhanced ambient occlusion
* Water reflection and refraction
* Auto-exposure
* Shader for all dimensions
* FAST!

## Tested on:
* Minecraft 1.16.x
* Nvidia and Intel
* Linux
